### FOUC Prevention Script (extend existing)

Location: `BaseLayout.astro` `<head>` inline `<script is:inline>`.
Current behavior: reads `localStorage.getItem('theme')`, sets `data-theme` if present.
Phase 4 extension: if no localStorage value, check `window.matchMedia('(prefers-color-scheme: light)')` вҖ” if matches, set `data-theme="light"`.

```js
// Extended FOUC script (Phase 4)
(function () {
  try {
    var stored = localStorage.getItem('theme');
    if (stored) {
      document.documentElement.setAttribute('data-theme', stored);
    } else if (window.matchMedia('(prefers-color-scheme: light)').matches) {
      document.documentElement.setAttribute('data-theme', 'light');
    }
    // No else branch вҖ” dark is the CSS default (no data-theme attribute needed)
  } catch (_) {}
})();
```

## Font Self-Hosting Contract

### Download source

Use Fontsource npm packages for clean, licensed WOFF2 files with proper unicode-range subsets:
- `@fontsource/space-grotesk` вҖ” extract weights 400, 500, 600
- `@fontsource/chakra-petch` вҖ” extract weights 400, 500, 600, 700
- `@fontsource/jetbrains-mono` вҖ” extract weight 400 regular + 400 italic

Alternative: Download direct from Google Fonts UI selecting Latin + Latin Extended only.

Files stored at: `public/fonts/*.woff2`
Naming convention: `{family}-{weight}-{style}.woff2` e.g. `space-grotesk-400.woff2`

### @font-face rules

Replace the Google Fonts `@import` in `src/styles/global.css` line 16 with `@font-face` blocks. Each block must include:
- `font-family`
- `src: url('/fonts/file.woff2') format('woff2')`
- `font-weight` (exact weight number)
- `font-style: normal` (or `italic`)
- `font-display: swap`
- `unicode-range: U+0000-00FF, U+0131, U+0152-0153, U+02BB-02BC, U+02C6, U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2122, U+2191, U+2193, U+2212, U+2215, U+FEFF, U+FFFD;` (Latin + Latin Extended вҖ” sufficient for PT-BR content)

### Preload link

Add to `BaseLayout.astro` `<head>`, before the FOUC script:

```html
<link rel="preload" href="/fonts/space-grotesk-400.woff2" as="font" type="font/woff2" crossorigin />
```

Only Space Grotesk 400 is preloaded вҖ” it is the critical path font for body text rendering. Other weights/families load async via `font-display: swap`.
