```astro
    <!-- Theme init: prevent FOUC by reading localStorage before paint -->
    <script is:inline>
      (function () {
        try {
          var stored = localStorage.getItem('theme');
          if (stored) {
            document.documentElement.setAttribute('data-theme', stored);
          } else if (window.matchMedia('(prefers-color-scheme: light)').matches) {
            document.documentElement.setAttribute('data-theme', 'light');
          }
          // No else branch — dark is the CSS default (no data-theme attribute needed)
        } catch (_) {}
      })();
    </script>
```

CRITICAL constraints:
- Keep `is:inline` attribute — this script must execute synchronously before first paint (FOUC prevention). Do NOT change it to a plain `<script>`.
- The dark mode is the CSS default (no `data-theme` attribute = dark). The script only sets `data-theme` when `stored` is present or system is light.
- Do NOT modify any other part of BaseLayout.astro.
