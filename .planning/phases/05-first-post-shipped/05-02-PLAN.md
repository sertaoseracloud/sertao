**Automated checks — run after deploy.yml green:**

```bash
SLUG="REPLACE_WITH_SLUG_FROM_05_01_SUMMARY"
BASE="https://sertaoseracloud.com"

# 1. Post is accessible
curl -s -o /dev/null -w "Post HTTP status: %{http_code}\n" "${BASE}/posts/${SLUG}"

# 2. canonical link tag present
curl -s "${BASE}/posts/${SLUG}" | grep -o '<link rel="canonical"[^>]*>'

# 3. JSON-LD BlogPosting present
curl -s "${BASE}/posts/${SLUG}" | grep -o '"@type":"BlogPosting"'

# 4. OG title tag present
curl -s "${BASE}/posts/${SLUG}" | grep -o '<meta property="og:title"[^>]*>'

# 5. OG description present
curl -s "${BASE}/posts/${SLUG}" | grep -o '<meta property="og:description"[^>]*>'

# 6. Twitter card present
curl -s "${BASE}/posts/${SLUG}" | grep -o '<meta name="twitter:card"[^>]*>'

# 7. Post in sitemap
curl -s "${BASE}/sitemap-index.xml" | grep -o "<loc>[^<]*</loc>" | head -5
curl -s "${BASE}/sitemap-0.xml" 2>/dev/null | grep "${SLUG}" || \
  curl -s "${BASE}/sitemap-1.xml" 2>/dev/null | grep "${SLUG}" || \
  echo "Checking sitemap pages — may need to try sitemap-0.xml or sitemap-1.xml"

# 8. Post in RSS feed
curl -s "${BASE}/rss.xml" | grep "${SLUG}"

# 9. Check Lighthouse CI run passed (last deploy.yml run)
gh run list --workflow=deploy.yml --limit=1 --repo sertaoseracloud/blog_sertao --json status,conclusion | node -e "const d=JSON.parse(require('fs').readFileSync('/dev/stdin','utf8')); console.log('Last run:', d[0]?.status, d[0]?.conclusion);"
```

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-07 | Denial of Service | Lighthouse CI testing real post URL | accept | Lighthouse CI is read-only (no POST requests to the site); GH Actions runner is ephemeral; no DoS vector. |
